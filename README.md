# md-2
A Java program to convert Markdown to HTML      
I finally started maintaining the [changelog](/CHANGELOG.md)!

## Usage
* Ensure that Java 21 is installed
* Run `md-2.jar`
* Alternatively, in the command line, run `java -jar md-2.jar`
